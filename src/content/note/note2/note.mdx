# @tailwindcss/typography-官方提供的typography插件

官方的typography插件，可以让我们在markdown中使用tailwindcss的样式，但是需要注意的是，它只能在markdown中使用，不能在html中使用。

## set of prose styles

使用範例:

```js
<BlogPost title={title}>
  <div
    class="p-10 prose
   prose-h1:text-default
   prose-h2:text-default
   prose-strong:text-danger
   prose-a:text-primary
   text-default
   bg-default max-w-none"
  >
    <Content />
  </div>
</BlogPost>
```

像是prose-strong:text-danger，prose-a:text-primary這樣的class，更改了以下內容中link與**text**的顏色
